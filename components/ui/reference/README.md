# UI Reference Designs

Saved reference implementations generated during Voyager V2 design phase.

## VoyagerInterface-v1-terminal.tsx

**Created:** 2025-12-26
**Source:** Gemini Flash + Claude refinement

The terminal-native conversation interface featuring:
- Context bar with environment variables ($CTX, $GRP)
- Tracer route visualization for graph traversal
- Astronaut avatar with terminal-dither filter
- Draft card with green tick approval pattern
- Command prompt input with hints

**View it:** `http://localhost:3000/preview`

**Key patterns established:**
- Obsidian background (#050505)
- Indigo accent for system elements
- Green for success/approval states
- SVG terminal-dither filter for images
- Monospace typography throughout
